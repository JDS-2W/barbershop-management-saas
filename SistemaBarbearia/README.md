# Sistema de Gestão para Barbearias

Aplicação web desenvolvida para gerenciamento de barbearias.

## Funcionalidades

- Agendamento de clientes
- Controle financeiro
- Painel administrativo
- Gestão de serviços

## Tecnologias

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- React

## Status do projeto

Projeto em desenvolvimento.
